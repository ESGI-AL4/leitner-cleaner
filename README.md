**Leitner Cleaner**

Pour récupérer le code:
```
git clone https://github.com/ESGI-AL4/leitner-cleaner
cd leitner-cleaner
checkout develop
```

Pour lancer le back:
```
cd backend/leitner-cleaner
npm install
npm run build
npm run start
```

Pour lancer les tests dans le back:
```
cd backend/leitner-cleaner
npm install
npm run test
```

Pour lancer la couverture de tests dans le back:
```
cd backend/leitner-cleaner
npm install
npm run test:cov
```
